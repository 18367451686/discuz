 Discuz! X 社区软件系统简介
+----------------------------------+
Crossday Discuz! Board（以下简称 Discuz!，中国国家版权局著作权登记号 2008SR11708
）是康盛创想(北京)科技有限公司（英文简称Comsenz）推出的一套通用的社区论坛软件系
统。作为国内最大的社区软件及服务提供商，Comsenz旗下的 Discuz! 产品，无论在功能、
稳定性、负载能力、安全保障等方面都居于国内外同类产品领先地位，是全球成熟度最高、
覆盖率最大的论坛软件系统之一。
